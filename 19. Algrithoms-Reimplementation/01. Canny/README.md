**Canny算法流程:**

灰度图像---->低通滤波去噪---->梯度计算---->使用sobel算子计算x和y方向的梯度，并将x和y方向的梯度做二范数---->(因为像素变化小的地方梯度值接近0)从而得到一张边缘图---->非极大值抑制(将强梯度值附近(按照梯度方向)的不太强梯度值的位置的值置为0)---->双阈值限定(大阈值以上的设为255，小阈值以下的设为0，大小阈值之间的如果紧邻大阈值以上像素，则设为255，否则设为0)
